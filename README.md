# AnimeTask 📝✨

AnimeTask é um gerenciador de tarefas desktop feito em Python com interface moderna inspirada em Notion e Todoist.

## ✨ Recursos
- Interface em cards moderna
- Tema escuro e claro
- Splash screen animada
- Sons ao interagir
- Estatísticas semanais
- Filtros e busca

## 🖥️ Tecnologias
- Python
- Tkinter
- Pillow

## 🚀 Como executar

```bash
python main.py
